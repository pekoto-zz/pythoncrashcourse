# Python Crash Course

## Strings

### Interpolation

_Python 3: Use f_

```python
name = 'Graham'
print(f'Hello, {name}')
```

_Python 2: Use .format()_

```python
first_name = 'graham'
last_name = 'mc'
print('Hello, {} {}'.format(first_name, last_name))
```

## Numbers

### Exponents

```python
print(2**3)
# prints 8
```

### Division

Integer division will __ALWAYS__ return a float.

```python
print(3/2)
# prints 1.5

```

## Constants

There is no constant type, but constants are conventionally written in all-caps.

```python
MAX_CONNECTIONS = 48
```

## Comments

Comments use #

```python
# This is a comment.
print('hi')
```
